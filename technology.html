2025年度　前期　テクノロジ　テスト対策

問題

問題1
PCの処理効率を高めるために、CPUが主記憶にアクセスする時間を見かけ上短縮することを目的としたものはどれか。

ア. SSD イ. 仮想記憶 ウ. キャッシュメモリ エ. デフラグ

解答: ウ. キャッシュメモリ
ポイント: キャッシュメモリは、CPUと主記憶（メインメモリ）の間に置かれる、非常に高速なメモリです。CPUがよく使うデータをキャッシュメモリに置いておくことで、低速な主記憶へのアクセス回数を減らし、処理を高速化します。

問題2
入力画面で数値を入力すべきところに誤って英字を入力したらエラーメッセージが表示され、再入力を求められた。このような工夫をしておく設計思想を表す用語として、適切なものはどれか。

ア. フールプルーフ イ. フェールソフト ウ. フォールトトレランス エ. ロールバック

解答: ア. フールプルーフ
ポイント: フールプルーフは、利用者が誤った操作をしても、あるいは誤った操作ができないようにすることで、システムの誤動作や危険な状況を防ぐ設計思想です。例：入力値のチェック、形状的に逆に差し込めないコネクタなど。

問題3
HTMLに関する記述のうち、適切なものはどれか。

ア. HTMLで記述されたテキストをブラウザに転送するためにFTPが使われる。 イ. SGMLの文法の基になった。 ウ. Webページを記述するための言語であり、タグによって文書の論理構造などを表現する。 エ. ＸＭＬの機能を縮小して開発された。

解答: ウ. Webページを記述するための言語であり、タグによって文書の論理構造などを表現する。
ポイント: HTML (HyperText Markup Language) は、Webページの構造を定義するための言語です。<h1>や<p>などの「タグ」で文章の各部分の意味（見出し、段落など）を示します。なお、Webサーバからブラウザへの転送にはHTTP/HTTPSが使われます。

問題4
インタフェース規格団体でないものはどれか。

ア. ANSI イ. IEEE ウ. EIA エ. XML

解答: エ. XML
ポイント: XML (eXtensible Markup Language) は、データの構造を記述するためのマークアップ言語の一つです。一方、ANSI（米国国家規格協会）、IEEE（米国電気電子学会）、EIA（米国電子工業会）は、工業製品などの規格を策定する団体です。

問題5
ミラーリング（RAID1）の説明として正しいものはどれか。

ア. 1つのデータをいくつかのハードディスクに分けて書き込むこと イ. 1つのデータを1つのハードディスクに書き込むこと ウ. いくつかのデータを1つのハードディスクに書き込むこと エ. いくつかのハードディスクに同じデータを書き込むこと

解答: エ. いくつかのハードディスクに同じデータを書き込むこと
ポイント: ミラーリング (RAID 1) は、2台以上のストレージに全く同じデータを同時に書き込むことで、信頼性を高める技術です。1台が故障しても、もう1台のデータで運用を継続できます。

問題6
パソコンA,Bからなる並列システムの稼働率として正しいものはどれか。なお、パソコンAの稼働率は0.9、パソコンBは0.8とする。

ア. 1.7 イ. 0.1 ウ. 0.72 エ. 0.98

解答: エ. 0.98
ポイント: 並列システムは、構成要素のいずれか一つが稼働していればシステム全体が稼働します。全体の稼働率は 1 - (すべての要素が故障する確率) で求められます。計算式は 1 - ((1 - 0.9) * (1 - 0.8)) = 1 - (0.1 * 0.2) = 1 - 0.02 = 0.98 となります。

問題7
プログラム言語ではないものはどれか。

ア. UX イ. Java ウ. Python エ. CoffeeScript

解答: ア. UX
ポイント: UX (User Experience) は、ユーザーが製品やサービスを通じて得られる体験や満足度のことを指す概念であり、プログラム言語ではありません。Java, Python, CoffeeScriptはコンピュータに命令を与えるためのプログラム言語です。

問題8
SuicaやPasmoで使われる、電波を使って無線通信するインタフェースはどれか。

ア. IrDA イ. RFID ウ. USB エ. SATA

解答: イ. RFID
ポイント: RFID (Radio Frequency Identification) は、ID情報を埋め込んだICタグと電波で無線通信し、情報を読み書きする技術です。交通系ICカードや商品の在庫管理などに利用されています。

問題9
MTBFの正式名称はどれか。

ア. Mean Time Between Failures イ. Mean Time Between False ウ. Mean Time Between Food エ. Mean Time Between Fourth

解答: ア. Mean Time Between Failures
ポイント: MTBF (Mean Time Between Failures) は「平均故障間隔」と訳され、システムが故障から復旧してから次に故障するまでの平均的な時間を示します。システムの信頼性を表す指標の一つです。

問題10
メカニカルな可動部分が無く、電力消費も少ないという特徴をもつ補助記憶装置はどれか。

ア. CD-RWドライブ イ. DVDドライブ ウ. HDD エ. SSD

解答: エ. SSD
ポイント: SSD (Solid State Drive) は、USBメモリなどと同じフラッシュメモリを記憶媒体として使用しているため、HDDのようなモーターやヘッドといった物理的な駆動部品がありません。そのため、衝撃に強く、読み書きが高速で、消費電力が少ないという利点があります。

問題11
電源供給が無くなるとデータが消える揮発性のメモリはどれか。

ア. DRAM イ. ROM ウ. SSD エ. USB

解答: ア. DRAM
ポイント: メモリは、電源を切ると内容が消える「揮発性メモリ」と、消えない「不揮発性メモリ」に大別されます。DRAM (Dynamic RAM) はコンピュータの主記憶（メインメモリ）として使われる代表的な揮発性メモリです。ROM, SSD, USBメモリは不揮発性です。

問題12
Webサイトを構築する際にCSS（Cascading Style Sheet）を用いる理由として、適切なものはどれか。

ア. WebサーバとWebブラウザ間で安全にデータをやり取りできるようになる。 イ. Webサイトの更新情報を利用者に知らせることができるようになる。 ウ. Webサイトの利用者を識別できるようになる。 エ. 複数のWebページの見た目を統一することが容易にできるようになる。

解答: エ. 複数のWebページの見た目を統一することが容易にできるようになる。
ポイント: CSSは、Webページのレイアウト、色、フォントなどの「見た目（スタイル）」を指定するための言語です。HTMLが文書の「構造」を、CSSが「装飾」を担当することで、効率的なWebサイト制作が可能になります。外部CSSファイルとして読み込むことで、サイト全体のデザインを一括で変更・管理できます。

問題13
通常使用されるメインPCと、そのメインPCの故障に備えて待機しつつほかの処理を実行しているサブPCの二つから構成されるコンピュータシステムはどれか。

ア. クライアントサーバシステム イ. デュアルシステム ウ. デュプレックスシステム エ. ピアツーピアシステム

解答: ウ. デュプレックスシステム
ポイント: デュプレックスシステムは、本番機（主系）と待機機（従系）の2系統で構成されます。待機機は普段は別の処理を行いつつ、本番機の故障時には処理を引き継ぎます。常に同じ処理を2台で行うのは「デュアルシステム」です。

問題14
フラッシュメモリではないものはどれか。

ア. SDカード イ. USBメモリ ウ. SSD エ. マスクROM

解答: エ. マスクROM
ポイント: フラッシュメモリは、電気的に内容を書き換え可能な不揮発性メモリの一種です。SDカード、USBメモリ、SSDはすべてフラッシュメモリを利用しています。マスクROMは、製造時にデータを書き込むもので、後から書き換えることはできません。

問題15
パソコンA,Bからなる直列システムの稼働率として正しいものはどれか。なお、パソコンAの稼働率は0.9、パソコンBは0.8とする。

ア. 1.7 イ. 0.1 ウ. 0.72 エ. 0.98

解答: ウ. 0.72
ポイント: 直列システムは、構成要素のすべてが稼働していないとシステム全体が停止します。全体の稼働率は、各要素の稼働率をすべて掛け合わせることで求められます。計算式は 0.9 * 0.8 = 0.72 となります。
